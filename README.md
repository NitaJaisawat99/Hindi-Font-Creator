# Hindi Font Creator 🎨

A fully free, fast, and powerful web app (PWA) to create stylish Hindi text, edit images, and download high-quality designs.

## 🌟 Features
- **10+ Stylish Hindi Fonts** (Calligraphy, Bold, Modern, etc.)
- **Multi-line Text Support** with Color & Opacity control
- **Canvas Background** (Solid Colors or Custom Image Upload)
- **Drag & Drop** to move, resize, and rotate text
- **High-Quality Export** (Save as PNG without watermark)
- **Direct Share** to WhatsApp, Instagram, etc.
- **Mobile Friendly** & Dark UI Theme
